User Adapter Application Node
=============================

    >>> from bda.bfg.ugm.model.users import Users
    >>> users = Users()
    >>> user = users['foo']
    >>> user
    <User object 'foo' at ...>

::

    >>> props = user.properties

User object is editable.
::

    >>> props.editable
    True

::

    >>> md = user.metadata
